# Brain Games — Colección de juegos CLI

Proyecto de práctica que reúne varios juegos interactivos de consola desarrollados en JavaScript (Node.js). Cada juego plantea pequeños retos matemáticos o lógicos y se juega desde la línea de comandos; el jugador responde a las preguntas que se muestran y el programa verifica las respuestas.

**Juegos incluidos:**
- `brain-games`: menú principal / lanzador de juegos
- `brain-even`: identificar si un número es par
- `brain-calc`: operaciones aritméticas simples
- `brain-gcd`: calcular el máximo común divisor
- `brain-progression`: completar la progresión aritmética
- `brain-prime`: identificar si un número es primo

**Uso rápido:**

1. Instalar dependencias:

```
npm install
```

2. Ejecutar un juego (ejemplo):

```
node bin/brain-even.js
```

Los badges de asciinema anteriores muestran grabaciones de ejecución para los juegos incluidos.
